# Prozedurale Sprachen - Zwischentests

This repository is a collection of all three solutions to the Zwischentests (interim tests / practice exercises) for my university course Prozedurale Sprachen (Procedural Languages). The intent of each Zwischentest was to monitor my progress in learning the C Language. A total of three Zwischentests were conducted.

With This repository you can review the solutions I made for the Zwischentests.

A description of the individual tasks can be found in the respective subfolder. The tasks were originally in German and the format was directly copied from the course, including all the tags. Merely the `<h5>` tags were replaced with `<h2>` tags and an `<h1>` tag was added at the top. Possible formatting issues may occur as a result of that. English translations are also provided, though there may be some errors.

**NOTE: The tasks for the Zwischentests vary for every batch, so my solution cannot and should not be copied and submitted. Instead, it may serve as a basis to see how one could approach such a task.**
